js CustomGestureEvent

설명

  • gesture 이벤트를 에뮬레이팅
  • CustomPointerEvent 를 상속 받는다

SPECIFICATION

event type

pointer event pointerdown pointermove pointerup pointercancel
custom pointer event custompointerdown custompointermove custompointerup custompointercancel
custom gesture event longpress longpresscancel
swipe, swipeleft, swiperight, swipeup, swipedown swipecacnel, swipeleftcacnel, swiperightcacnel, swipeupcacnel, swipedowncacnel
pinch, zoom, pinchzoom
rotate

event static property

  • CustomGestureEvent.longPressTimeout : longpress 이벤트의 발생 시간 (ms). 기본값 500 (0.5초)
  • CustomGestureEvent.longPressCancelThreshold : longpress 진행 중 포인터 이동 시 취소되는 임계점. 기본값 10px
  • CustomGestureEvent.swipeVelocityThreshold : swipe 동작시 필요한 속도. 기본값 300(px/sec)
  • CustomGestureEvent.pinchZoomDistanceThreshold : pinch, zoom 발생의 임계점. 0.1px
  • CustomGestureEvent.rotateThreshold : rotate 발생의 임계점. 0rad (제한 안함)

event options

  • bubbles : true
  • cancelable : true
  • composed : true
  • detail : event.detail()

event.detail = customPointerEvent.detail 과 같다.
