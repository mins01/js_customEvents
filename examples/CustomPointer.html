js CustomPointer

설명

  • pointer event 기반으로 pointer 정보 가공/제공

SPECIFICATION

constructor

  • customPointer =new CustomPointer(event)

method

  • null = CustomPointer::setFirst(event) // 최초 이벤트 정보를 저장. 생성자에서 알아서 호출한다.
  • null = CustomPointer::setEvent(event) // 현재 이벤트를 연결.
  • number (px) = CustomPointer::distanceBetween(customPointer) // 다른 커스텀 포인터와의 거리
  • number (rad) = CustomPointer::angleBetween(customPointer) // 다른 커스텀 포인터와의 각도

property

  • int = CustomPointer::pointerId // event.pointerId
  • bool = CustomPointer::isPrimary // event.isPrimary
  • object = CustomPointer::first // 최초 이벤트 정보(pageX,pageY,timeStamp)
  • Event = CustomPointer::event // 현재 이벤트
  • number (ms) = CustomPointer::duration // 활동 시간
  • number (px) = CustomPointer::distanceX // X축 이동 거리
  • number (px) = CustomPointer::distanceY // Y축 이동 거리
  • number (px) = CustomPointer::distance // 이동 거리
  • number (px/ms) = CustomPointer::velocityX // X축 이동 속도 (이동 거리/활동 시간)
  • number (px/ms) = CustomPointer::velocityY // Y축 이동 속도 (이동 거리/활동 시간)
  • number (px/ms) = CustomPointer::velocity // 이동 속도 (이동 거리/활동 시간)
